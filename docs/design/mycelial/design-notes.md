## 1. 概念内核

标签不是数据库的行，而是**会生长的活网络**。Taxon 管理的就是一张活的标签有机体，所以**界面本身应当是这个有机体**，而不是把它塞进内容区当一张图。设计语言从产品本质（taxonomy = 网络 / 生长 / 共现）长出来，不是贴上去的装饰。

## 2. 色彩系统（提亮暖调）

早期近黑底太冷、丢了"有机温暖"。锁定版提亮加暖：

```css
:root{
  --soil:#221a12;   /* 暖腐殖土底 */
  --soil2:#2c2317;  /* 次级表面 */
  --cream:#f1e9da;  /* 主文字 */
  --dim:#ab9e89;    /* 次文字 */
  --bio:#6ff5c8;    /* 生物荧光 teal —— 主强调/维度节点 */
  --bio2:#c4f85a;   /* 孢子 lime —— 高频/活跃 */
  --amber:#eaa066;  /* 暖土 —— 跨维共现/警示 */
  --thread:rgba(111,245,200,.34); /* 菌丝线 */
  --glass:rgba(28,22,15,.5);      /* 悬浮 chrome 玻璃 */
  --sheet:rgba(30,24,16,.72);     /* 密集页玻璃 sheet */
}
```

**大气**：苔藓绿（左下）+ 琥珀（右上）光晕叠在提亮腐殖土上 —— 暗处发光，不是纯黑。
原则：荧光三色只做**点状强调**，绝不大面积铺。

## 3. 字体

Hanken Grotesk（人文无衬线）+ Noto Sans SC（思源黑体，CJK）+ Space Mono（等宽标签/编号）。

## 4. 标志母题：标签有机体

力导向菌丝网络 —— 标签是发光菌体（大小=使用量），共现是有机生长曲线（`stroke-dashoffset` 生长动画），节点辉光呼吸（`glow` 无限 alternate）。**取代通用饼图/折线图**，是产品的视觉签名。

## 5. 布局：画布优先（核心决策）

**抛弃传统"边栏 + 顶栏 + 内容区"** —— 那副后台模板骨架与"活网络"母题自相矛盾。锁定范式：

- **有机体即界面**：标签网络铺满整个视口，是主体
- **悬浮 chrome**：导航 / 状态 / 数据读数都是漂浮在有机体之上的**半透明 HUD**，无实心边栏/顶栏
- **导航即菌丝**：左侧一串发光节点 + 菌丝连线；每个节点**带图标**（静止可辨）+ **hover 滑出标签**（救可发现性）；激活节点常显标签
- **HUD 数据读数**：KPI 作为发光数字漂浮在四角，非底栏

### 分层原则（关键）

| 页面类型 | 布局 |
|---------|------|
| **英雄页**（仪表盘、实体图谱） | 画布优先，有机体主导，悬浮 HUD |
| **密集页**（检索、审核、治理表） | 内容置于**半透明玻璃 sheet**，浮于（暗化+模糊的）活画布之上；nav spine 保留 |

密集页范式见 `mockup-canvas-search.html`：画布退后（`opacity:.4;filter:blur`），数据 sheet（`--sheet` + `backdrop-filter:blur(16px)`）浮于其上。**密集数据也不退回笨重边栏**——你始终在有机体里，只是聚焦了一次查询。

## 6. 动效

菌丝生长（stroke-draw）+ 节点辉光呼吸 + 孢子脉动。克制，有"生长感"，必带 `prefers-reduced-motion`。

## 7. 落地映射（Tailwind / 目标技术栈）

目标 `packages/console`（Next 16 + React 19 + Tailwind）。§2 token 写入 `globals.css`；字体经 `next/font/google`；有机体用 SVG + 绝对定位节点（样张即参考实现），与实体图谱 #100/#101 共用一套渲染。

## 8. 待办 / 风险

- 画布优先在密集页的 sheet 可再让更多画布"透气"（当前 sheet 偏大偏实）
- 荧光对比度需过 WCAG（`--cream` on `--sheet` OK；`--dim` 小字慎用）
- 落地必须**先清结构债**（#104 仪表盘溢出 / #106 hydration），别在裂布局上换皮

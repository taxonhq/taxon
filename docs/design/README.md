# Taxon 设计语言探索

把 Taxon 从"干净但平庸的通用 SaaS 后台"升级为有创意度、有品牌人格的产品。核心创意母题：**taxonomy（分类学）本身**——层级、网络、聚类、共现、生长——取代通用饼图/折线图，成为产品的视觉签名。

2026-05 探索了四个方向（星图 / 标本馆 / 菌丝 / 终端），每个都把"分类"翻译成一个独立的视觉世界。各方向预览为单文件 HTML，浏览器打开即看动效。

## 方向归档

| 方向 | 状态 | 内核 | 目录 |
|------|------|------|------|
| **菌丝 Mycelial** | ✅ **已选为主方向** | 标签是会生长的活网络；暗色 + 生物荧光 | [`mycelial/`](./mycelial/) |
| **标本馆 Herbarium** | 📚 存档参考 | 林奈博物学；分类即标本图版 | [`herbarium/`](./herbarium/) |
